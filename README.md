
umuus-google-oauth
==================

Installation
------------

    $ pip install umuus_google_oauth

Usage
-----

    $ python -m umuus_google_oauth run         --credential_file 'client_secret_XXXX.apps.googleusercontent.com.json'         --token_file google_access_token.json


Example
-------

    $ umuus_google_oauth

    >>> import umuus_google_oauth
