
umuus-google-oauth
==================

Installation
------------

    $ pip install umuus_google_oauth

$ python umuus_google_oauth.py run     --credential_file 'config/google/client_secret_XXXX.apps.googleusercontent.com.json'     --token_file /tmp/o.json

Example
-------

    $ umuus_google_oauth

    >>> import umuus_google_oauth
